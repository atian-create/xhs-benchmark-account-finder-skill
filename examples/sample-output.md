# Sample Benchmark Account Finder Output

## Input

```text
Track: AI tools for solo creators
Follower target: 3k-30k
Format: graphic notes and screen-recording tutorials
Goal: find accounts I can learn from without copying them
```

## Output Shape

Recommend 3 main benchmark accounts if evidence is available. For each one, show track fit, stage fit, recent update signal, learnable parts, non-copyable parts, and the next note type to study.

## Boundary Note

This sample uses user-provided material and public-safe judgment only. It does
not require platform login, automatic publishing, private analytics connectors,
or copying another creator's content.
